Materi Singkat



Cara Kerja Internet

Secara singkat, proses internet dimulai dari klien meminta alamat Google http yang diminta ke server disebut dengan http request. Server akan mencarikan menggunakan DNA server. Server kemudian melakukan copy paste ke kita sebagai respon yang dikenal sebagai cookies. Ini menerima copyan halaman web yg dikopikan server ke kita. Agar kita ga proses lama lagi disebut dengan http response (hasil di komputer kita)


Bahasa Internet

Terdapat tiga bahasa internet yang dipelajari yaitu HTML, CSS, JS.

  • HTML merupakan komponen internet yang diartikan sebagai sebuah objek. HTML mengatur content dari suatu web.
  • CSS merupakan komponen internet yang diartikan sebagai properties. CSS mengatur tampilan/presentasi dari web.
  • JS merupakan komponen internet yang diartikan sebagai suatu aksi action. JS mengatur bagaimana suatu objek bergerak.


WebGIS dari sisi Client dan Server

Client adalah perangkat yang melakukan akses terhadap website (orang/manusia yang mengakses juga dapat disebut client). Client side yaitu bekerja dengan html, java script, memanggil data yg tersimpan dalam server. Client hanya menampilkan hasil yang diproses dalam server.


Server adalah perangkat keras maupun lunak komputer yang berfungsi untuk menyimpan halaman web pada alamat tertentu. Server side yaitu diawali web broser, melakukan panggilan menggunakan salah satu http lalu php, execute, lalu muncul misalnya "Hello World". Server menyimpan banyak data yang disimpan dalam big data.


Bootstrap dan LeafletJS

Bootstrap menyediakan komponen yang sudah 'dibungkus' untuk memudahkan pembangunan web menggunakan HTML, CSS dan JS. Web yang dibangun dengan Bootstrap bersifat Responsive, artinya dapat dengan mudah menyesuaikan ukuran pada browser desktop atau mobile.

LeafletJS merupakan library atau kumpulan fungsi berbasis Javascript yang digunakan untuk menampilkan peta interaktif pada halaman web. Leaflet menyediakan Map API (Application Programming Interface) yang memudahkan web developer untuk menampilkan peta berbasis Tile pada halaman web. Pengguna peta juga dapat berinteraksi dengan menggunakan fungsi telah disediakan oleh Leaflet.
